Principales algoritmos de manejo de cadenas en lenguajes de programación

1. Python

Cadenas inmutables (str). Usa internamente arreglos dinámicos. Métodos: len(), +, find(), replace(), split(), join(), upper(), strip()

2. Java

String inmutable, StringBuilder mutable. Métodos: length(), concat(), substring(), indexOf(), replace(), split()

3. C++

std::string (mutable). Operadores: +, +=, []. Métodos: length(), substr(), find(), append(), replace()

4. JavaScript

Primitivo inmutable. Métodos: length, concat(), slice(), indexOf(), replace(), split(), toUpperCase(), trim()

5. C#

string inmutable, StringBuilder mutable. Métodos similares a Java.

6. PHP

Cadenas tratadas como arrays de bytes. Funciones: strlen(), substr(), strpos(), str_replace(), explode(), implode(), trim(), strtoupper()

Algoritmos clásicos de búsqueda de patrones

  • Naive (fuerza bruta) – O(m·n)
  • KMP (Knuth-Morris-Pratt) – O(m+n) con tabla de prefijos
  • Boyer-Moore – muy rápido en texto largo
  • Rabin-Karp – usa hashing
